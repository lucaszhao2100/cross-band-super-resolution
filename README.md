This dataset is designed for research in cross-spectral image super-resolution and includes scripts, model training code, and multi-band image data. It supports tasks such as data preprocessing, model evaluation, and spectral enhancement. The dataset is well-structured to facilitate reproducibility and is dynamically updated by the creator to reflect ongoing improvements in algorithms, data quality, and implementation. It is intended for use in remote sensing, hyperspectral image analysis, and machine learning applications related to image super-resolution.
