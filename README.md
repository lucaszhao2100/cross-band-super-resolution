🌟 Project Overview
This dataset is specifically designed for research in cross-spectral image super-resolution. It provides a comprehensive collection of multi-band remote sensing image data, along with complete scripts for data preprocessing, state-of-the-art model training code, and standardized evaluation tools.
The dataset is meticulously structured to ensure full reproducibility of experiments and is dynamically maintained by the creator to incorporate the latest advancements in algorithms, data quality improvements, and implementation optimizations. It serves as a valuable resource for researchers and practitioners in the fields of remote sensing, hyperspectral image analysis, and machine learning.
